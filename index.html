Information:

"Wai Not?" is a widget add-on that is built on top of the Makahiki open source project. It combines various game aspects to create a fun learning environment.

Current Development:

Currently the development has not been progressed much. Most of the project is still under construction. See the Gameplay documentation to view what has been implemented so far..
